## Comment Api

### `GET` Comments
```
/api/posts/{post}/comments
```
Get list all product's comments
#### Request Headers
| Key | Value |
|---|---|
|Accept|application/json

#### Query Param
| Param | Type | Description |
|---|---|---|
| post | int | Get comments's post (require) |
| perpage | int | number item per page |
| page | int | paginate comments |
